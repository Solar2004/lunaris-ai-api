# Lunaris AI API Documentation

Welcome to the Lunaris AI API. This API provides a unified interface for interacting with multiple AI providers.

## API Endpoints

- **[Chat](./endpoints/chat.md)** (`POST /chat`)
  - Standard conversational AI interface.
  
- **[Completions](./endpoints/completions.md)** (`POST /completions`)
  - Code autocompletion with Fill-In-The-Middle (FIM) support.

- **[Models](./endpoints/models.md)** (`GET /models`)
  - List available models and providers.
